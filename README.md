# Nede.JS logging from zero to expert

## Running

```bash
# Install dependencies
yarn install

# Fill environment file
cp .env.dist .env

# Running app
yarn run start
```

## Request

```bash
curl http://0.0.0.0:3000/api/v1/posts/1
```
